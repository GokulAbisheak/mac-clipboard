# Security Policy

## Supported Versions

Security fixes are provided for the latest release only.

## Reporting a Vulnerability

If you discover a security issue, please do not open a public issue.

Use one of these private channels:

- Open a private GitHub Security Advisory for this repository.
- If advisories are not enabled, contact the maintainer directly and include:
  - A clear description of the issue
  - Reproduction steps or proof-of-concept
  - Impact assessment
  - Suggested remediation (if available)

You can expect:

- An initial acknowledgment within 7 days
- A follow-up after triage with severity and next steps
- Coordinated disclosure after a fix is available
